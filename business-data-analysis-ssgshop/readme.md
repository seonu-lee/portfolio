# SSG SHOP 상품 분류 자동화 및 사이즈 적정성 분석

## 프로젝트 개요
본 프로젝트는 온라인 편집숍 **SSG SHOP**의 콘텐츠 운영 효율 개선을 목표로 수행되었다.  
대규모 입고 예정인 Black / Blue 계열 의류 상품을 대상으로 머신러닝 기반 자동 분류 모델을 구축하고,  
셔츠 상품의 브랜드별 사이즈 불일치 문제를 통계적으로 분석하여 고객 클레임 감소를 위한 인사이트를 도출한다.

본 프로젝트는 **컴퓨터 비전 기반 분류 문제**와 **통계적 가설 검정 문제**를 하나의 시나리오 안에서 다루며,  
단순 모델 비교를 넘어 실무 적용 가능성까지 고려한 분석을 목표로 한다.

---

## Task 1. 이미지 기반 의류 상품 자동 분류

### 문제 정의
SSG SHOP은 Black / Blue 계열의 의류 상품을 대량으로 입고할 예정이며,  
수작업 분류 방식은 시간·비용·오분류 리스크 측면에서 한계를 가진다.  
이에 따라 5개 상품 카테고리(dress, pants, shirt, shoes, shorts)를 자동으로 분류하는 모델을 구축한다.

### 접근 방법
- **Baseline 모델**
  - 색 히스토그램 + HOG 특징 추출
  - Logistic Regression, Linear SVM 비교
- **Advanced 모델**
  - ImageNet 사전학습된 ResNet-18 기반 CNN 전이학습
- 클래스 불균형 문제를 고려하여 WeightedRandomSampler 적용

### 모델 성능 비교
| 모델 | Accuracy | Macro F1 | 특징 |
|---|---|---|---|
| Logistic Regression | 0.67 | 0.64 | 계산 효율적, 해석 용이 |
| Linear SVM | 0.58 | 0.56 | 수렴 불안정 |
| CNN (ResNet-18) | 0.97 | 0.97 | 형태·구조 학습 가능 |

CNN 모델은 전통적 머신러닝 대비 약 **50% 이상의 성능 향상**을 보였으며,  
의류 이미지 분류에서는 색상 정보만으로는 한계가 있고 형태·패턴을 학습하는 딥러닝 접근이 효과적임을 확인하였다.

### 결론
실무 적용 관점에서 CNN 기반 자동 분류 모델이 신규 상품 분류에 가장 적합한 대안임을 제안한다.

## Task 2. 셔츠 사이즈 적정성 및 브랜드 간 차이 분석

### 분석 배경
고객만족팀 VOC에 따르면, 동일한 사이즈 표기(S, M, L 등)임에도 브랜드별 실제 착용감 차이로 인한  
반품 및 클레임이 빈번하게 발생하고 있다. 이를 데이터 기반으로 검증한다.

### 분석 질문
1. 동일 Brand Size 내에서도 브랜드별 실제 치수 차이가 존재하는가?
2. 셔츠 소재(Type)는 실제 치수에 영향을 미치는가?
3. 특정 두 브랜드 간 동일 사이즈 기준 실제 치수 차이가 존재하는가?

### 분석 방법
- 기술통계 및 시각화(Boxplot, Stripplot)
- One-way ANOVA
- Brand Size 통제 후 다중회귀 분석
- 독립표본 t-test

### 핵심 결과
- 동일 사이즈 표기(L, XL)라도 브랜드 간 실제 치수 차이는 통계적으로 유의함
- 소재 효과는 단순 비교에서는 유의하지 않으나, Brand Size를 통제하면 유의미한 영향 확인
- 특정 브랜드 쌍(WROGN vs English Navy)은 가슴둘레보다 **어깨너비에서 더 큰 차이**를 보임

### 해석
사이즈 표기만으로는 실제 핏을 예측하기 어려우며,  
소비자 경험 개선을 위해 브랜드·소재별 상세 치수 정보 제공이 필요함을 시사한다.
